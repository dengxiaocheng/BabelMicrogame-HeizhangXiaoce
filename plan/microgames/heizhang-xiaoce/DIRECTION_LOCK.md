# Direction Lock: 黑账小册

## One Sentence
玩家用暗号记录黑账，写得越清楚越有用，也越容易被搜出。

## Core Loop
1. 执行核心循环：选择事件 -> 编码记录 -> 藏匿 -> 经历搜查 -> 最终解码证据
2. 按 20 分钟节奏推进：基础记录 -> 搜查 -> 编码变复杂 -> 解码关键证据

## Must Keep
- 核心机制必须保持：证据编码 + 藏匿风险 + 解码难度
- 核心循环必须保持：选择事件 -> 编码记录 -> 藏匿 -> 经历搜查 -> 最终解码证据
- 20 分钟结构只作为节奏，不扩成大项目

## Must Not Add
- 不做笔记软件；核心是证据清晰度和安全
- 不新增第二套主循环
- 不把小游戏扩成长期经营或开放世界

## Required State
- evidence_clarity
- stash_risk
- encoding_difficulty
- search_pressure
- round

## Success
在限定时长内完成主循环，并稳定进入至少一个可结算结局

## Failure
关键状态崩溃，或在本轮主循环中被系统淘汰
