# 黑账小册

Babel microgame source repository.

## 试玩

在线试玩：[https://dengxiaocheng.github.io/BabelMicrogame-HeizhangXiaoce/](https://dengxiaocheng.github.io/BabelMicrogame-HeizhangXiaoce/)

## 本地运行

```bash
# 用任意静态服务器打开 index.html（需要支持 ES modules）
npx serve .
# 或
python3 -m http.server 8000
```

然后在浏览器打开对应地址即可。

## 测试

```bash
npm test
```
